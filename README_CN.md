## 特性

- 灵活的由应用、插件、模块组成
- 通用抽象的数据系统
- 面向接口和面向数据编程
- 事件驱动和数据驱动
- 数据和过程追踪
- 面向策划人员友好，使用 Excel 作为配置文件
- 较低的学习和培训成本
- 基于标准 C++开发，更容易学习和使用
- 跨平台(Windows, Linux, MacOS)
- 高可用的系统架构
- 高并发和高性能的网络模块
- 配套的 U3D 客户端
- 丰富的插件(DB, script, HTTP, WebSocket 等)
- 提供商业定制服务
